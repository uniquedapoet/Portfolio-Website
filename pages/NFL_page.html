About the Project

This project is a NFL offensive Analysis done for the final prject of my 'Python for Information Science' course. This analysis was done using NFL api's as well as datasets from Kaggle. The analysis was done using Python with the main goal to discover the most productive offense in the NFL. The analysis was done using the following metrics and wieghts: Score:3x, First Downs:2x, Yards Per Game:2x, Possesion:1x. After the project was turned in I decided to train a linear model to predict the weighted score of each team. I wouldnt rely on the model to make any decisions but it was a fun project to work on putting together all the skills I learned in future courses.
