About My Time At Situs

During the Summer of 2024, I served as a Data Engineer/Analyst intern at SitusAMC in their Denver office. In collaboration with another intern, I assisted our team in developing a new data pipeline. My primary responsibility was creating a Rust-based parser to translate a specialized internal SQL dialect into standard SQL, which streamlined data processing and reduced manual workloads. Additionally, we partnered with clients to develop a comprehensive loan processing system. This system utilized SQL queries to aggregate data, amortize loans, and generate Standard Monthly Mortgages (SMMs), thereby supporting strategic decision-making and enhancing operational efficiency.

Key Contributions

  • Developed a Rust-Based Parser: Translated a niche internal SQL dialect into standard SQL, improving data processing efficiency and minimizing manual interventions.
  • Data Pipeline Development: Collaborated on building a robust data pipeline that ensured seamless data flow and integration across systems.
  • Loan Processing System: Worked with clients to design and implement a system for processing loans, leveraging SQL for data aggregation, loan amortization, and generation of Standard Monthly Mortgages (SMMs).
  • Predictive Analysis: Conducted predictive analyses to forecast loan performance, aiding in risk assessment and strategic planning.
