About the Project

This project is a stock market trading model that when trades sp500 stocks, the model uses a combination of technical indicators and machine learning algorithms to predict stock price movements and generate trading signals. The model is designed to identify profitable trading opportunities by analyzing historical price data, volume, and other market variables. The model is built to trade on one stock at a time to maximize profits and minimize risk. After simulating trading on a few stocks year to date, the model has shown an average of 12% profit.

The model is built using Python and various libraries, including Pandas, NumPy, Scikit-learn, and TensorFlow. The model is trained on historical stock price data and uses a combination of technical indicators, such as moving averages, RSI, and MACD, to generate trading signals. The model is then evaluated on a test set to assess its performance and accuracy.
